# VSCODE

 VSCODE 사용하는 단축키 , 사용하는 extension 정리

***
### Shortcut
  - 작업 장소 이동 (alt + <>)
  - 검색창 열기 (ctrl + shift + f)
  - 문서(에디터) 닫기 (ctrl + f4, ctrl+w)
  - 새 창 (ctrl + shift + n)
  - 새 탭 (ctrl + n)
  - 단축키 세팅 (ctrl + k + s)
  - 창 분할 (ctrl + \\)
  - 내어쓰기 (shift + tab)
  - 줄 삭제 (ctrl + shift + k)
  - 포커싱 줄 이동(alt + 화살표)
  - 포커싱 줄 복사(shift + alt + 화살표)
  - 라인 이동 (ctrl + g)
  - 괄호 끝부분 이동 ( ctrl + shift + \\)
  - 블럭 주석 (shift + alt + a)
  - 한줄, 범위 주석 (ctrl + /)
  - 자동 정렬 (ctrl + k + f)
  - 문자열 n개 수정(ctrl + d)
  - 다음 오류로 이동 (f8)
  - 파일 찾기 (ctrl + p)
